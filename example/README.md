# Fabricate — example app

Quick local playground for the package, including the floating Stub Studio UI.

## Run

```bash
cd example
npm install
npm run dev
```

Then open the URL printed by Vite (usually http://localhost:5173).

PO/QA guide page:

- http://localhost:5173/po-qa-guide.html

## What to test

- click `Seed demo users`
- click `POST /users` / `GET /users`
- open the `Stub Panel` drawer
- click `PO/QA guide` from the panel header
- try scenario presets and live field mapping edits
