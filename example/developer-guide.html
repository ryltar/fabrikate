Fabricate Developer Guide

Learn how to implement Fabricate in your code: integrate the client, create stubs, build test scenarios, and configure field generation for localStorage-backed testing.

Quick start

Create a typed client

import { createFetchClient } from "fabricate"; type User = { id: number; name: string; role: "admin" | "user"; }; const client = createFetchClient({ baseUrl: "https://api.example.com", headers: { Authorization: "Bearer token", }, });

Call the API

const user = await client.get<User>("/users/123"); const newUser = await client.post<User>("/users", { name: "Alice", role: "user", });

Enable stub mode

const client = createFetchClient({ baseUrl: "https://api.example.com", stub: true, // Enable stub mode devMode: true, // Enable UI and manager });

Using Fabricate Studio

Mount the UI

import { createFetchClient, mountStubStudioDrawer } from "fabricate"; const api = createFetchClient({ stub: true, devMode: true, }); mountStubStudioDrawer(api, { launcherLabel: "Stub Panel", position: "bottom-right", initiallyOpen: false, widthPx: 600, });

View storage

// In Fabricate Studio Storage tab: localStorage: { "fabricate:collection:/users": [ { "id": "u-1", "name": "Ada", "role": "admin" }, { "id": "u-2", "name": "Grace", "role": "user" }, ], "fabricate:resource:/profile": { "id": "me", "firstName": "John", "role": "admin", }, }

Manage field mappings

// In Fabricate Studio Mappings tab: JSON config: { "/users": { "age": { "type": "number.int", "min": 18, "max": 65 }, "displayName": { "type": "person.fullName" }, "createdAt": { "type": "date.recent", "days": 10 }, }, } // When you POST /users, fields auto-generate const newUser = await api.post("/users", { name: "Bob" }); // Result: { name: "Bob", age: 42, displayName: "John Doe", ... }

Run scenarios

// Scenarios are buttons in Fabricate Studio // Click to instantly populate localStorage or reset state const api = createFetchClient({ stub: true, devMode: true, stubScenarioPresets: createScenarios(), });

Stubbing in code with Fabricate Core

Set up test data

const prefix = api.stubManager.getStoragePrefix(); // Store collection data api.stubManager.setStorageValue( `${prefix}:collection:/users`, [ { id: "u-1", name: "Ada", role: "admin", }, { id: "u-2", name: "Grace", role: "user", }, ] ); // Store resource data api.stubManager.setStorageValue( `${prefix}:resource:/profile`, { id: "me", firstName: "John", role: "admin", } );

Define scenarios

// example/src/scenarios.js export function createScenarios() { return { happyPath: { label: "Happy Path Demo", description: "Load realistic test data", execute(manager) { manager.clearStorage(); const prefix = manager.getStoragePrefix(); manager.setStorageValue( `${prefix}:collection:/users`, [{ id: "u-1", name: "Ada", role: "admin" }] ); }, }, emptyState: { label: "Empty State", description: "Clear all data", execute(manager) { manager.clearStorage(); manager.setPostGeneratedFieldsJson(undefined); }, }, }; }

Configure field generation

const api = createFetchClient({ stub: true, stubPostGeneratedFieldsJson: { "/users": { age: { type: "number.int", min: 18, max: 65, asString: true, }, displayName: { type: "person.fullName" }, createdAt: { type: "date.recent", days: 10 }, category: { type: "pick", values: ["admin", "user", "guest"], }, }, }, }); // When you POST, fields auto-generate const newUser = await api.post("/users", { name: "Bob", }); // Result includes: age, displayName, createdAt, category

Clear storage and save state

// Save state before testing const snapshot = api.stubManager.exportStorageSnapshot(); // ... run your test operations ... await api.post("/users", { name: "Test User" }); // Clear and restore api.stubManager.clearStorage(); api.stubManager.importStorageSnapshot(snapshot, { clearFirst: true, });

Collection CRUD examples

// GET collection - returns array const users = await api.get<User[]>("/users"); // GET item by id const user = await api.get<User>("/users/u-1"); // POST - creates new item const newUser = await api.post<User>("/users", { name: "Bob", role: "user", }); // PATCH - updates item await api.patch<User>("/users/u-1", { role: "admin", }); // DELETE - removes item await api.delete("/users/u-1");

Resource examples

// GET resource - returns stored object const profile = await api.get<Profile>("/profile"); // PATCH - merges updates into stored object const updated = await api.patch<Profile>("/profile", { firstName: "Jane", }); // PUT - replaces entire resource await api.put<Profile>("/profile", { id: "me", firstName: "NextName", role: "user", });
